# Adventure.IO
### A JavaScript Library for Text Adventures

The intention of Adventure.IO is to create an easy to use, easy to deploy and easy to test Text Adventure library for JavaScript.

Features: 
* None! I'm working on it

Coming soon:
* Verbs
* Rooms
* Inventory
* Audio Cues!
